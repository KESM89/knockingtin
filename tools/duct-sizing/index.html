Standards & Method

  • Equal Friction Method — duct sized so friction loss is constant per unit length throughout the system. Design friction rate 0.1" WG per 100 ft per SMACNA and ASHRAE Fundamentals.
  • Round diameter — calculated using ASHRAE Fundamentals formula: Hf = 0.109 × Q¹·⁹ / D⁵·⁰², solved for D. Result rounded up to nearest standard size.
  • Rectangular equivalents — calculated using ASHRAE hydraulic diameter equivalence: De = 1.30 × (a×b)⁰·⁶²⁵ / (a+b)⁰·²⁵. Width rounded up to nearest even inch.
  • Velocity limits (SMACNA/Manual D) — Residential supply: 600–900 FPM main, 400–700 FPM branch. Commercial supply: 1,000–1,500 FPM main, 600–1,000 FPM branch. Return air: 10–20% lower than supply.
  • Standard sizes — round duct in 1" increments 4"–24", then 26", 28", 30". Rectangular widths rounded to nearest even inch.
